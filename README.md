# WasteRoute-Env 🚛

An OpenEnv reinforcement learning environment simulating urban waste collection. An AI agent controls a garbage truck navigating a city road network, deciding which bins to collect while managing fuel efficiently.

## Real-World Motivation

Municipal waste collection is a genuine logistics challenge faced by cities worldwide. Poor routing wastes fuel, increases emissions, and causes bin overflow. This environment lets agents learn optimal collection strategies on a graph-based city model.

## Environment Description

- **World:** City represented as a weighted graph (nodes = locations, edges = roads with distances)
- **Agent:** Controls a garbage truck starting at depot (node 0)
- **Goal:** Collect waste bins efficiently while managing limited fuel

## Action Space
```json
{
  "action_type": "move" | "collect",
  "target_node": int,
  "message": "optional text for LLM agents"
}
```

## Observation Space
```json
{
  "current_node": int,
  "bin_levels": {"node_id": fill_level_0_to_1},
  "fuel_remaining": float,
  "collected_bins": [int],
  "step_count": int,
  "total_reward": float,
  "done": bool,
  "message": str,
  "graph_edges": [[from, to, distance]]
}
```

## Tasks

| Task | Bins | Fuel | Dynamic | Max Steps | Difficulty |
|------|------|------|---------|-----------|------------|
| easy | 5 | 1.0 | No | 30 | ⭐ |
| medium | 15 | 0.7 | No | 60 | ⭐⭐ |
| hard | 20 | 0.5 | Yes | 80 | ⭐⭐⭐ |

## Reward Function

- `+fill_level` for collecting a bin ≥ 70% full
- `+fill_level × 0.3` for collecting a bin < 70% full
- `-0.3` for revisiting already collected bin
- `-0.05` per move step (fuel efficiency)
- `-0.5` per bin overflow (hard task only)
- `-1.0` when fuel runs out

## Graders (0.0 – 1.0)

- **Easy:** Collection rate × 0.6 + quality × 0.4 + fuel bonus
- **Medium:** Collection × 0.5 + quality × 0.3 + fuel efficiency × 0.2
- **Hard:** Collection × 0.5 + quality × 0.3 + fuel × 0.2 - overflow penalty

## Baseline Scores

| Task | Score |
|------|-------|
| easy | 0.540 |
| medium | 0.344 |
| hard | 0.301 |
| **average** | **0.395** |

*Baseline agent: Qwen/Qwen2.5-72B-Instruct via HF Inference API*

## Setup & Usage

### Connect to deployed environment
```python
import requests

# Reset
response = requests.post(
    "https://prasooon-wasteroute-env.hf.space/reset",
    json={}
)
obs = response.json()

# Step
response = requests.post(
    "https://prasooon-wasteroute-env.hf.space/step",
    json={"action": {"action_type": "move", "target_node": 1}}
)
```

### Run locally
```bash
git clone https://huggingface.co/spaces/Prasooon/wasteroute-env
cd wasteroute-env
pip install openenv-core fastapi uvicorn pydantic openai
uvicorn server.app:app --host 0.0.0.0 --port 7860
```

### Run baseline inference
```bash
export HF_TOKEN="hf_..."
export API_BASE_URL="https://router.huggingface.co/v1"
export MODEL_NAME="Qwen/Qwen2.5-72B-Instruct"
python inference.py
```

### Docker
```bash
docker build -t wasteroute-env:latest .
docker run -p 7860:7860 wasteroute-env:latest
```
